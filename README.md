# 业余兴趣项目

这里是我的个人开发项目存储地，包含各种技术实验和兴趣项目。

## 📁 目录结构

```
业余兴趣/
├── .claude/                    # Claude AI 配置
├── .github/                    # GitHub 配置文件
├── projects/                   # 项目集合目录
│   └── README.md               # 项目索引说明
├── news-analyzer/              # 新闻分析器项目
│   ├── src/                    # Vue 3 前端源码
│   ├── src-tauri/              # Rust 后端源码
│   ├── .gitignore              # Git 忽略文件配置
│   ├── package.json            # Node.js 依赖配置
│   └── README.md               # 项目文档
└── README.md                   # 本文件
```

## 🔧 环境要求

- Node.js (v16+)
- Rust (最新稳定版)
- Git

## 📋 项目快速开始

每个项目都有独立的运行方式，请查看对应项目的 README.md 文件。

## 🗂️ 项目管理

此目录遵循以下组织原则：
- 扁平化结构，避免过深嵌套
- 每个项目独立存放
- 配置文件统一管理
- 定期清理无用文件和依赖
